# Kwezy Buses Website Redesign
## HCI Comprehension Project - Problem Three

**Student ID:** 22311351007  
**Tool:** Figma  
**Project Duration:** November 20 - December 3, 2025

## Project Overview
Redesigning the Kwezy Buses website (https://www.kwezybuses.com) to improve usability, accessibility, and user experience for bus ticket booking in Malawi.

## Original Interface
- **Type:** Bus booking website
- **Primary Function:** Display bus routes and pricing
- **Target Users:** Travelers in Malawi booking intercity bus tickets

## Identified Problems (Preliminary)
1. Information overload with 23+ routes displayed simultaneously
2. No search or filter functionality
3. Poor mobile responsiveness
4. Inconsistent text formatting
5. No clear booking pathway
6. Redundant route listings
