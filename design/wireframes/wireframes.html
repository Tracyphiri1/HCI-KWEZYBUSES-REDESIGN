üìã HCI Design Principles Applied
  • Visibility of System Status: Progress indicators, seat availability, selected items highlighted
  • User Control & Freedom: Clear back buttons, ability to modify search, cancel bookings
  • Consistency: Uniform button styles, consistent layout patterns, standard icons
  • Error Prevention: Seat selection visual feedback, form validation, clear CTAs
  • Recognition over Recall: All options visible, no hidden menus, visual seat map
  • Flexibility: Filters for power users, simple search for beginners
  • Aesthetic & Minimalist: Only essential information shown, white space utilized
  • Help Users Recover: Clear error messages (in final design), cancellation policy visible
